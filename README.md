## 🚀 Projects

* **[SpotifyMirror](https://github.com/Mint3Ds/SpotifyMirror)** – A backend automation tool that seamlessly mirrors Spotify playlists to YouTube by creating equivalent YouTube playlists with matching tracks.

* **[EnergiKita](https://github.com/Mint3Ds/EnergiKita)** *(Hackathon Finalist)* – A full-stack web application that helps households improve energy efficiency by analyzing electricity bills and appliance usage, providing personalized recommendations to reduce electricity consumption and costs.

* **[Internship Result Management System](https://github.com/Mint3Ds/Internship-Result-Management-System)** – A full-stack web application developed as a university coursework project. It streamlines internship assessment by allowing lecturers and industry supervisors to evaluate students, while providing administrators with tools to manage authorized users and oversee the evaluation process.

* **FishingAutomation** *(Private Repository)* – A Python automation script built with OpenCV (`cv2`) that detects in-game visual cues to automate fishing actions, demonstrating practical applications of computer vision and automation.
